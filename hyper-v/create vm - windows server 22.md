# Setup A Virtual Machine

In this doc, we'll setup a VM using Hyper-V.

We will use an ISO for Windows Server 2022.

In our right hand menu, we'll click **New > Virtual Machine**

<img width="354" height="348" alt="image" src="https://github.com/user-attachments/assets/3dc37b67-19c1-44b4-b33e-edf657ea6e50" />

This will launch the New Virtual Machine Wizard and give it a name:

<img width="704" height="533" alt="image" src="https://github.com/user-attachments/assets/b34ee39e-8d82-41a0-b42f-be87fe955603" />

Next, we'll choose between Generation 1 and Generation 2 virtual machines

*Gen 1 emulates and uses older, legacy firmware. Gen 2 uses modern firmware for newer systems*

I'll choose Gen 2:

<img width="704" height="533" alt="image" src="https://github.com/user-attachments/assets/0ea6670c-2497-4fdd-8663-891906618f6d" />

Let's assign memory:

<img width="519" height="164" alt="image" src="https://github.com/user-attachments/assets/41836ff3-b75b-4fc0-bfaa-d5a8b0c07193" />

Next, let's configure which switch we will use:

<img width="525" height="122" alt="image" src="https://github.com/user-attachments/assets/65a502fc-cbbf-4e3f-ab53-df05a9be989b" />

Next, we will create the Virtual Disk:

<img width="532" height="370" alt="image" src="https://github.com/user-attachments/assets/b7890d9f-1ff5-479b-8ca3-f8ab24d1583e" />

Now, we'll select the ISO image we want to use.

I'll use my Windows Server 2022 ISO:

<img width="514" height="212" alt="image" src="https://github.com/user-attachments/assets/29eba636-6dd0-4815-bd03-db1a0898807a" />

Let's look over our summary and click 'Finish'
