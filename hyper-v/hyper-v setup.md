# Setting Up Windows Hyper-V

In this doc, we will install Windows Hyper-V, which is used for Virtualization.

*This is only available on Windows Pro or Enterprise versions*
*Your computer must also support Virtualization as well*

We'll go to **Control Panel > Programs and Features > Turn Windows features on or off**
