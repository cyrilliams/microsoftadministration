# Setting Up Windows Hyper-V

In this doc, we will install Windows Hyper-V, which is used for Virtualization.

*This is only available on Windows Pro or Enterprise versions*
*Your computer must also support Virtualization as well*

We'll go to **Control Panel > Programs and Features > Turn Windows features on or off**

<img width="1125" height="634" alt="image" src="https://github.com/user-attachments/assets/135fc1aa-52a7-4412-9abb-b60634b8525b" />

We'll enable both the Management Tools and Platform and click 'OK'.

<img width="415" height="368" alt="image" src="https://github.com/user-attachments/assets/523ff9bf-52d6-4281-9b10-c90dc12384a0" />

After a reboot, we'll find out Hyper-V Manager

<img width="760" height="713" alt="image" src="https://github.com/user-attachments/assets/02e3475f-9be3-455b-96ac-bf7445df8de5" />

Now we have our Hyper-V Manager open and the ability to create a new virtual machine, virtual switch or SAN.
