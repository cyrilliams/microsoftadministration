# Defender Cloud App Policies

Microsoft Defender allows you to create and use policies to manage your cloud apps.

We can go to **Defender > Cloud apps > Policies > Policy Management**

Here we can see all of our Cloud App Policies:

<img width="1614" height="770" alt="image" src="https://github.com/user-attachments/assets/7341c0d5-097e-4508-8ccf-7e71e2734901" />

## Create a Cloud App Policy from Policy Templates

If we wanted to create a policy from a template, 
we can go to: **Defender > Cloud apps > Policies > Policy Templates**

We can see templates that we can use to create our own templates.
Let's click the + next to one:

<img width="1587" height="828" alt="image" src="https://github.com/user-attachments/assets/4376d82f-43d3-4b92-8b33-f24a1669aa6d" />

We can set the policy template, name, severity, category and we can set when the policy acts.

<img width="721" height="819" alt="image" src="https://github.com/user-attachments/assets/37db191f-fe62-4916-b7c2-1a2e2c82dd8a" />

Single activity will enforce the policy on each instance.

Repeated activity allows the policy to repeat a set amount of times before enforcing.

We can also configure the filters for how when the policy enforces and set alerts:

<img width="1019" height="709" alt="image" src="https://github.com/user-attachments/assets/dd5734b2-4d78-4c78-bde4-0623e3838481" />

Now in our Policies, we can see our new Policy has been added:
